Keylogger – Projet personnel

Passionné par la sécurité informatique, j’ai décidé de créer un keylogger en Python pour mieux comprendre comment les données sensibles, comme les frappes clavier, peuvent être interceptées. Le but de ce projet était d'explorer les concepts de surveillance et de capture de données dans un cadre sécurisé et éthique.

Le keylogger utilise la bibliothèque keyboard pour enregistrer toutes les frappes clavier de l'utilisateur. Les informations sont ensuite envoyées via des sockets à un serveur distant. Ce système permet de récupérer en temps réel les frappes sans que l’utilisateur s'en aperçoive. J’ai utilisé cette approche pour étudier comment les données peuvent être volées dans un environnement vulnérable et comment sécuriser les systèmes contre ce genre d’attaque.
